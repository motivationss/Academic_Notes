\begin{abstract}
	This course will use Hatcher\cite{hatcher2002algebraic} as the main text, but the order may differ here and there. Enjoy this fun
	course! In particular, I add some extra content which is not covered in lectures, things like \hyperref[def:groupoid]{groupoid},

	the text as much as possible, but I may still miss some.

	\par This course is taken in Winter 2022, and the date on the covering page is the last updated time.
\end{abstract}
